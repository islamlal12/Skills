# Brand Identity Expert Skill

A specialized skill designed to maintain the "Islam - AI Ecom Expert" personal brand. It provides a single source of truth for persona alignment, visual guidelines, and strategic market positioning for the Algerian E-commerce sector.

## Installation

### Repository Only
Files are located in `.agents/skills/brand-identity/`.

### Global Installation
To install globally:
```bash
ln -sf "e:/Agents Automation/.agents/skills/brand-identity" "~/.claude/skills/brand-identity"
```

## Usage

This skill triggers when you need to:
- Write copy for reels, stories, or ads targeting Algerian store owners.
- Design dashboards or landing pages with a consistent "Islam" aesthetic.
- Define service bundles and pricing for automation systems.

## Directory Structure

- `SKILL.md`: Core logic and brand summary.
- `references/`: Detailed guides for voice, tone, and technical standards.
- `resources/`: Design tokens and assets.
- `examples/`: Code and content samples.
- `scripts/`: Brand-related utility scripts.
